Обновление BIOS для видеокарт KFA2!

Мы внесли ряд изменений после детального изучения поступивших предложений от владельцев видеокарт KFA2. Новая версия BIOS в том числе содержит обновленные алгоритмы управления вентиляторами. Используйте новый BIOS только в том случае, если на вашей системе замечено нетипичное поведение видеокарты.

Обновление BIOS рекоммендуется, но не является обязательным. Не забудьте сделать бэкап текущей версии BIOS.

Шаги для обновления:

  1. Загрузить соответствующую программу обновления для вашей графической карты KFA2.
  2. Извлечь всё из загруженного архива в любую папку.
  3. Перезагрузить компьютер. После загрузки, не запуская иных программ, перейти к шагу 4.
  4. Запустить извлечённую программу обновления.
  5. После завершения обновления, компьютер будет перезагружен автоматически.

* Программа обновления предназначена для запуска в операционных системах Windows 7/ 8/ 10
